# Game Concept: IRONCLAW

*Created: 2026-03-29*
*Status: Draft*

---

## Elevator Pitch

> 포스트 아포칼립스 황무지에서 건블레이드를 휘두르며 방사능 돌연변이를 사냥하는
> 탑다운 슈터 ARPG. 소울나이트의 직관적 원터치 조작에 디아블로2의 중독적 루팅과
> 스킬 변형 시스템을 결합했다. 20분 런으로 밀도 있게, 유니크 아이템 하나로 빌드
> 전체가 뒤집히는 득템의 쾌감.

## Core Fantasy

황무지의 생존자로서, 점점 강력한 하이브리드 무기와 장비를 갖춰가며
한때 두려웠던 돌연변이들을 압도적으로 학살하게 되는 **성장의 쾌감**.

처음엔 거대 변이 고양이 한 마리에 쫓기던 캐릭터가, 유니크 건블레이드에
화염 속성을 실어 돌연변이 무리를 한 번에 소각하는 존재로 성장한다.
그 과정에서 떨어지는 장비 하나하나가 새로운 가능성을 열어준다.

---

## Unique Hook

소울나이트 같은 탑다운 슈터인데, **디아블로2처럼 유니크 아이템이 스킬 자체를
변형시키고**, 칼과 총이 결합된 **건블레이드**로 근접-원거리 콤보가 하나의
무기에서 터진다.

건블레이드는 이 세계관의 상징적 무기 — 기술이 쇠퇴한 세계에서 냉병기와 화기를
결합한 실용적이고 야만적인 발명품. 베고, 찌르고, 방아쇠를 당기면 화약이 터지며
추가 데미지. 이 전투감이 다른 모바일 슈터와 차별화되는 핵심.

## Player Experience Analysis (MDA Framework)

### Target Aesthetics (What the player FEELS)

| Aesthetic | Priority | How We Deliver It |
| ---- | ---- | ---- |
| **Sensation** (sensory pleasure) | 3 | 타격감, 이펙트, 드랍 사운드, 화면 흔들림 |
| **Fantasy** (make-believe) | 4 | 포스트 아포칼립스 황무지 사냥꾼 정체성 |
| **Narrative** (drama) | 7 | 최소한의 환경 스토리텔링, 보스 배경 정도 |
| **Challenge** (mastery) | 2 | 던전 난이도 스케일링, 보스 패턴 |
| **Fellowship** (social) | N/A | 싱글 플레이 |
| **Discovery** (exploration) | 5 | 유니크 아이템 효과 발견, 빌드 시너지 탐색 |
| **Expression** (self-expression) | 3 | 클래스 + 아이템 + 스킬 조합으로 나만의 빌드 |
| **Submission** (comfort zone) | 6 | 파밍 런의 반복적 쾌감 |

### Key Dynamics (Emergent player behaviors)

- 유니크 아이템 효과를 보고 스킬 조합을 실험한다
- "이 던전에서 이 유니크가 나온다"는 정보를 바탕으로 특정 던전을 반복 파밍한다
- 쿨다운 감소 스택을 쌓아 스킬 난사 빌드를 만든다
- 건블레이드 폭발 타이밍을 자동 콤보 안에서 최적화한다
- 지역별 몬스터 속성에 맞춰 장비를 교체한다

### Core Mechanics (Systems we build)

1. **탑다운 슈터 전투** — 자동 조준 + 건블레이드 자동 콤보 (근접-사격 체인)
2. **4등급 아이템 시스템** — 노멀/매직/레어/유니크, 등급별 옵션 차등
3. **스킬 변형 시스템** — 유니크 아이템이 스킬에 속성/효과를 추가
4. **던전 생성** — 지역별 방 풀에서 랜덤 조합, 몬스터 강도 스케일링
5. **캐릭터 성장** — 레벨업 + 장비 파밍 + 스킬 강화 (영구 보유)

## Player Motivation Profile

### Primary Psychological Needs Served

| Need | How This Game Satisfies It | Strength |
| ---- | ---- | ---- |
| **Autonomy** (freedom, meaningful choice) | 5개 클래스 선택, 빌드 자유도, 던전 선택 | Supporting |
| **Competence** (mastery, skill growth) | 장비 강화로 수치적 성장 + 보스 패턴 학습으로 실력 성장 | Core |
| **Relatedness** (connection, belonging) | 세계관 몰입, 클래스 정체성 | Minimal |

### Player Type Appeal (Bartle Taxonomy)

- [x] **Achievers** (goal completion, collection, progression) — 아이템 수집, 등급 올리기, 전 던전 클리어, 빌드 최적화
- [x] **Explorers** (discovery, understanding systems) — 유니크 아이템 효과 발견, 스킬 시너지 실험, 숨겨진 조합 탐색
- [ ] **Socializers** — 해당 없음 (싱글 플레이)
- [ ] **Killers/Competitors** — 해당 없음 (PvP 없음)

### Flow State Design

- **Onboarding curve**: 첫 던전은 튜토리얼 겸용. 이동 → 기본 공격 → 스킬 1개 순차 해금. 10분 내에 코어 루프 체험 완료
- **Difficulty scaling**: 몬스터 강도가 플레이어 진입 시 상태(레벨, 장비) 기반 스케일링. 항상 약간의 도전감 유지
- **Feedback clarity**: 데미지 숫자 팝업, 크리티컬 이펙트, 레벨업 연출, 유니크 드랍 시 특별 이펙트/사운드
- **Recovery from failure**: 죽으면 마을로 복귀, 페널티 없음. 즉시 재도전 가능. 좌절보다 "다음엔 된다" 느낌

## Core Loop

### Moment-to-Moment (30 seconds)

던전 방 진입 → 가상 조이스틱으로 이동 (자동 조준) → 건블레이드 자동 콤보로
몬스터 처치 (베기-베기-쏘기 체인) → 스킬 버튼으로 액티브 스킬 사용 (쿨다운 기반)
→ 몬스터 처치 시 아이템/골드 드랍 → 즉시 획득

핵심: 이 30초가 **아이템 없이도 재미있어야 한다.** 건블레이드 콤보의 타격감,
처치 시 이펙트, 회피의 긴장감이 기본 재미를 보장.

### Short-Term (5-15 minutes)

던전 층 탐색: 일반 방 (전투) → 특수 방 (상점/보물/이벤트) → 보스 방 → 보상
→ 다음 층 또는 귀환 선택

각 지역은 다양한 방 셋에서 랜덤 선택되어 매 런마다 다른 구성.
"이번엔 어떤 방이 나올까" + "보스에서 유니크가 떨어질까"가 긴장감 유지.

### Session-Level (40-60 minutes)

마을에서 장비 정리/교체 → 던전 1런 (20분) → 마을 복귀, 보상 확인 →
장비 비교/교체 → "이 유니크 끼고 다시 돌아볼까..." → 던전 2런 (20분)

자연스러운 정지점: 런 종료 후 마을에서. 하지만 새 장비를 끼면 "한 판만 더"
심리가 작동.

### Long-Term Progression

- 캐릭터 레벨업 → 스탯 성장 + 새 스킬 해금
- 새 던전 지역 해금 (레벨/스토리 기반)
- 더 높은 등급 아이템 드랍 풀 확장
- 유니크 아이템 수집 → 새로운 빌드 가능성 확장
- 5개 클래스 육성 (각각 다른 빌드 경험)
- 엔드게임: 고난이도 던전, 레어 유니크 파밍

### Retention Hooks

- **Curiosity**: "다음 지역엔 어떤 유니크가 있을까?", "이 스킬에 빙결 옵션 붙이면?"
- **Investment**: 키운 캐릭터, 모은 장비 컬렉션, 해금한 클래스
- **Social**: 해당 없음 (싱글 플레이)
- **Mastery**: 보스 패턴 학습, 빌드 최적화, 더 높은 난이도 도전

## Game Pillars

### Pillar 1: "한 방의 쾌감" (Loot Dopamine)
아이템 한 개가 빌드 전체를 바꿀 수 있어야 한다. 득템의 순간이 게임의 최고 순간.

*Design test*: "이 유니크 아이템이 흥미롭긴 한데 밸런스가 걱정된다" →
**넣는다.** 밸런스보다 "이거 미쳤다!"라는 반응이 우선.

### Pillar 2: "20분의 밀도" (Dense Runs)
한 런이 20분 안에 끝나되, 빈 시간 없이 전투-보상-선택이 빽빽해야 한다.

*Design test*: "이 시스템이 깊이를 주지만 런 시간이 30분으로 늘어난다" →
**빼거나 줄인다.** 모바일 세션 밀도가 우선.

### Pillar 3: "한 손의 몰입" (Effortless Control)
자동 조준 + 4버튼으로 복잡한 빌드가 직관적으로 작동해야 한다.
조작이 고민되는 순간은 없어야 한다.

*Design test*: "수동 조준을 넣으면 스킬 천장이 높아진다" →
**안 넣는다.** 조작 복잡도보다 접근성이 우선.

### Pillar 4: "황무지의 색" (Gritty World)
포스트 아포칼립스의 거칠고 기괴한 분위기가 무기, 몬스터, 던전 모든 곳에
일관되게 스며들어야 한다.

*Design test*: "깔끔하고 판타지풍 디자인이 더 대중적이지 않을까?" →
**아니다.** 그릿하고 독특한 세계관이 차별점이다.

### Anti-Pillars (What This Game Is NOT)

- **NOT 복잡한 조작 게임**: 깊이는 빌드에서 나오지 조작에서 나오지 않는다. 가상 패드 6개, 수동 조준 같은 건 없다.
- **NOT 스토리 중심 게임**: 세계관은 분위기를 위한 것. 긴 대사나 컷신으로 루프를 끊지 않는다.
- **NOT PvP 게임**: 싱글 플레이 파밍 루프에 집중. 멀티플레이어는 스코프를 폭발시킨다.

## World Setting

### 세계관 배경
대규모 핵전쟁 이후, 문명이 붕괴된 세계. 방사능 오염으로 동식물이 돌연변이화되었고,
살아남은 인간들은 기술 쇠퇴로 냉병기에 의존하게 되었다. 남아있는 화약 기술과
냉병기를 결합한 **건블레이드**류 하이브리드 무기가 이 시대의 상징.

극한의 상황은 이단 종교, 미친 과학자, 군벌 등 다양한 인간 세력을 낳았다.

### 무기 컨셉: 하이브리드 무기
- **건블레이드 (Gunblade)**: 검 + 화약 방아쇠. 베기 후 발사로 폭발 추가 데미지
- **건슬라이서 (Gunslicer)**: 대형 도 + 산탄. 횡베기 시 산탄 확산
- **스팅샷 (Stingshot)**: 창/작살 + 라이플. 찌르고 빵 쏘면 관통
- 기타 변형: 도끼+화약, 메이스+폭발, 활+장약 등 세계관에 맞는 야만적 결합

### 던전 지역 (5개)

**1. 폐허 도시 (Ruined City)**
무너진 빌딩과 잔해 사이. 돌연변이 인간, 변이 개, 거대 변이 고양이(쌍두),
변이 사슴이 서식. 가장 접근하기 쉬운 초반 지역.

**2. 방사능 늪지 (Irradiated Swamp)**
독안개와 오염된 물웅덩이. 거대 양서류, 변이 파충류, 작살 든 늪지 인간.
늪에 맞는 특수 장비(늪지 신발 등)가 드랍되는 지역.

**3. 지하 연구소 (Underground Lab)**
깨진 유리 배양관, 비상등 불빛. 매드 사이언티스트와 종간 혼합 키메라 실험체.
가장 기괴한 몬스터가 등장하는 지역.

**4. 붕괴 군사기지 (Collapsed Base)**
벙커, 녹슨 장갑차, 무너진 격납고. 훈련된 군인과 군사 병기화된 돌연변이.
가장 전술적인 전투가 벌어지는 지역.

**5. 이단 성소 (Heretic Sanctum)**
뼈로 장식된 원시적 제단, 야생적인 의식 공간. 식인을 신성시하는 광신도와
의식용 변이체. 인육을 먹는 종교라는 설정에 맞게 와일드하고 불쾌한 분위기.

### 보스 디자인 원칙
- **이성 유지형**: 극도로 강한 피지컬 + 독특한 무기. 전략적 패턴, 대화 가능.
- **변이 괴물형**: 거대하고 흉폭한 돌연변이. 압도적 존재감, 이성 상실.

### 몬스터 디자인 원칙
- 방사능 돌연변이가 기본 테마 (거대화, 다두, 키메라 등)
- 지역별 고유 몬스터 + 인간형 적은 전 지역 공통
- 팔레트 스왑으로 변종 생성 (녹색 = 독, 붉은색 = 광폭화 등)

## Class System

5개 클래스. 전통적 역할군이지만 포스트 아포칼립스 세계관에 맞는 독특한 네이밍.

| 역할 | 클래스명 (가안) | 컨셉 | 주력 무기 |
| ---- | ---- | ---- | ---- |
| 전사 | **Ironclad** | 중장갑 근접 전문, 돌파형 | 대형 건블레이드, 건메이스 |

## Item System

### 등급 체계 (Diablo II 기반)

| 등급 | 색상 | 옵션 수 | 특징 |
| ---- | ---- | ---- | ---- |
| **Normal** | 흰색 | 0-1개 | 기본 스탯만. 초반 사용, 후반 재료 |
| **Magic** | 파란색 | 1-2개 | 랜덤 접두/접미 옵션 |
| **Rare** | 노란색 | 3-4개 | 다중 랜덤 옵션, 높은 수치 |
| **Unique** | 주황색 | 고정 옵션 + 특수 | **스킬 변형 옵션** 포함. 빌드를 정의하는 아이템 |

### 유니크 아이템의 스킬 변형 예시

> **[화염의 건블레이드]** — Unique 무기
> - 기본 공격력 +50
> - 훨윈드 스킬에 화염 데미지 추가
> - 화염 피해를 받은 적은 마지막 타격 이후 3초간 초당 화염 데미지 지속
> - 타격할 때마다 지속 데미지 타이머 리셋
>
> → 훨윈드를 계속 맞추면 화염 DoT가 유지되면서 추가 데미지 누적.
>    빌드의 핵심이 되는 아이템.

### 아이템 옵션 풀 (예시)
- 공격력/방어력 증가
- 쿨다운 감소
- 이동 속도 증가
- 특정 속성 데미지 추가 (화염, 빙결, 독, 전기)
- 체력 흡수
- 크리티컬 확률/데미지
- 특정 스킬 레벨 +N (유니크 전용)
- 스킬 속성 변환 (유니크 전용)

## Skill System

### 구조
- 클래스당 6-8개 스킬 (해금식)
- 동시 장착: 3개 (스킬 슬롯)
- 제약: 쿨다운 기반 (마나 없음)
- 강화: 레벨업으로 기본 강화 + 유니크 아이템으로 속성/효과 변형

### 스킬 변형 메카닉 (유니크 아이템 연동)
유니크 아이템이 장착된 스킬에 추가 효과를 부여:
- **속성 추가**: 화염/빙결/독/전기 데미지 추가
- **행동 변경**: 범위 확대, 투사체 추가, 지속시간 변경
- **시너지**: 특정 조건 충족 시 보너스 (예: "빙결된 적 타격 시 폭발")

쿨다운 감소가 아이템 옵션으로 등장 → "스킬 난사 빌드" 가능.
마나 시스템 없이도 충분한 빌드 다양성 확보.

## Controls

### 모바일 조작 레이아웃
- **왼쪽**: 가상 조이스틱 (이동)
- **오른쪽**: 버튼 4개
  - 기본 공격 (자동 콤보: 베기-베기-사격 체인)
  - 스킬 1, 2, 3 (쿨다운 표시)
- **자동 조준**: 가장 가까운 적 자동 타겟
- **무기**: 캐릭터 옆에 떠있는 별도 스프라이트, 조이스틱 방향으로 회전

### 비주얼 스타일 (소울나이트 참조)
- 캐릭터: 심플한 스프라이트 (정면/측면), 최소 애니메이션
- 무기: 캐릭터 주변에 떠있는 별도 스프라이트, 방향 회전
- 공격: 무기 흔들림 + 이펙트로 표현 (별도 공격 모션 불필요)
- 클래스 구분: 실루엣/색상 차이로 구분

## Risks and Open Questions

### Design Risks
- 아이템/스킬 조합 밸런스가 폭발할 수 있음 (유니크 아이템이 많아질수록)
- 자동 조준 + 자동 콤보가 너무 수동적으로 느껴질 수 있음 — 스킬 사용 타이밍이 깊이를 줘야 함
- 로그라이트 구조에서 "천장"에 도달하면 파밍 동기가 사라질 수 있음

### Technical Risks
- 솔로 개발에서 Godot 모바일 빌드 최적화 경험 부족
- 아이템 옵션 랜덤 생성 시스템의 데이터 구조가 복잡해질 수 있음
- 에셋 일관성 유지 (여러 소스에서 구매 시)

### Market Risks
- 모바일 탑다운 슈터 시장에 소울나이트가 이미 강력한 선점자
- "디아블로 깊이"가 모바일 캐주얼 유저에게는 진입장벽이 될 수 있음

### Scope Risks
- 유니크 아이템 50+개 각각의 스킬 변형 효과 설계/구현 볼륨
- 5개 던전 지역의 방 셋 + 몬스터 + 보스 = 콘텐츠 제작 볼륨

### Open Questions
- 건블레이드 자동 콤보의 "감각"이 실제로 재미있는가? → MVP 프로토타입으로 검증
- 쿨다운만으로 충분한 스킬 리소스 관리 깊이가 나오는가? → 프로토타입으로 검증
- 에셋 스타일 통일을 어느 수준까지 달성할 수 있는가? → 에셋 조사 필요
- 모바일 성능에서 파티클 이펙트 + 다수 몬스터 동시 처리가 가능한가? → 벤치마크 필요

## MVP Definition

**Core hypothesis**: "건블레이드 자동 콤보 전투 + 디아블로식 아이템 드랍이
20분 런에서 '한 판 더'를 유발하는가?"

**Required for MVP**:
1. 클래스 1개 (Ironclad), 기본 공격 + 스킬 2개
2. 건블레이드 자동 콤보 전투 (베기-베기-사격 체인)
3. 던전 1개 (폐허 도시, 방 8-10종 랜덤 구성), 보스 1체
4. 아이템 4등급 시스템 작동 (등급별 2-3종)
5. 유니크 아이템 1-2개 (스킬 변형 효과 포함)
6. 마을(로비) — 장비 교체만 가능
7. 플레이스홀더 아트 (컬러 도형 + 기본 이펙트)

**Explicitly NOT in MVP**:
- 나머지 4개 클래스
- 나머지 4개 던전 지역
- 아이템 강화/제작 시스템
- 상점/NPC 시스템
- 최종 아트 에셋
- 사운드/음악

### Scope Tiers

| Tier | Content | Features | Timeline |
| ---- | ---- | ---- | ---- |
| **MVP** | 클래스 1, 던전 1, 아이템 10종 | 코어 루프만 | 4-6주 |
| **Vertical Slice** | 클래스 2, 던전 2, 아이템 20종 | 코어 + 마을 + 장비 비교 | 2-3개월 |
| **Alpha** | 클래스 3, 던전 3, 아이템 40종 | 전 시스템 러프 | 4-6개월 |
| **Full Vision** | 클래스 5, 던전 5, 유니크 50+ | 완성, 폴리시 | 8-12개월 |
